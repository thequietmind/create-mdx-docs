# Getting Started

## Adding Pages

Create a new `.mdx` file in the `pages/` directory, then register it in `config/pages.js`:

```js
import { lazy } from "react";

const MyPageMDX = lazy(() => import("@pages/my-page.mdx"));

export const pages = [
  // ...existing pages
  {
    name: "My Page",
    route: "/my-page",
    component: MyPageMDX,
  },
];
```

## Using Components

Import any React component at the top of an MDX file, then use it inline in your content.

[Material UI](https://mui.com/material-ui/) is already installed, so you can import any of its components:

```mdx
import { Button } from "@mui/material";

<Button variant="contained" color="primary">
  Click me
</Button>
```

Your own components work the same way — import them with a relative path from the `.mdx` file:

```mdx
import { Callout } from "../src/components/Callout.jsx";

<Callout type="tip">
  Register new pages in `config/pages.js`.
</Callout>
```

## Site Config

Update `config/site.js` to set your site name and description:

```js
export const site = {
  name: "My Docs",
  description: "My project documentation",
};
```
