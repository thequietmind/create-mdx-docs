# Welcome

This is your new documentation site, powered by [MDX Docs](https://mdxdocs.com).

Write your content in MDX — a format that lets you use React components alongside Markdown.

## What is MDX?

MDX lets you use JSX in your markdown content. You can import components and use them inline:

```jsx
import { Button } from "@mui/material";

// My Page
<Button variant="contained">Hello</Button>
```

## Next Steps

- Add new pages in the `pages/` directory
- Register them in `config/pages.js`
